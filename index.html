Web Hosting

Una web hosting es el servicio donde se alojan los contenidos necesarios para una página web (texto,imágenes,datos, enclaces...) aciéndola accesible a cualquier usuario de internet en el momento que sea.

En este caso se ha usado el github como entorno de desarrollo en el cual muchos creadores web confían en él para guardar sus proyectos y así poder compartirlos. En mi caso la dirección web que tengo en github es lorena10tl.github.io
